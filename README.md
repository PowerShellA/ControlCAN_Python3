ControlCAN二次封装后的Python3调用(封装)
# CAN硬件接口封装 
# 绘制图形所需数据的处理
# Mustangta

#### 介绍

CAN数据采集与展示的CAN的硬件接口封装部分 
包括C++的DLL库初始化和库函数的调用，用Python3进行了Python的封装。
旨在抛弃晦涩的硬件参数和变量以对硬件进行方便的调用。
为数据显示的方便，再次封装了ioface，能将数据以list形式的帧返回
